## Getting Started

### Installing

#### **create and edit server config file in res/config.json**
```
vim config/server.json
```
```json
{
	"local_server": "url of api on serverside.",
	"server": "url of api over network."
}
```

### Compiling

#### **transile the source into javascript**

```
yarn build
```

### Running

#### **run the program**

```
yarn start
```
